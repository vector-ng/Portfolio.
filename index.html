Web Project — Independent Build

VectorVerse

A live Web3 market intelligence platform built from scratch — tracking 24+ tokens in real time, aggregating crypto news from 20+ RSS sources, and generating branded post graphics via a custom Canvas engine. Engineered with React and Next.js, powered by Vercel serverless functions that proxy CoinGecko and global APIs to handle CORS and deliver live data. Fully deployed with SEO, PWA configuration, and Open Graph metadata.

What I worked on
  • Built and deployed full-stack platform using React, Next.js, and Vercel
  • Engineered serverless API proxy functions for real-time token and news data
  • Developed Canvas-based post graphic generator for dynamic branded image creation
  • Implemented SEO, PWA, and Open Graph metadata for discoverability
